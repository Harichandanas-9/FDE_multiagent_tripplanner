# Multi-Agent Trip Planner

LangGraph-style multi-agent system with FastAPI chat UI, PII guardrail, and a built-in evaluation dashboard.

## Run

```
pip install -r requirements.txt
python run.py
```

Open <http://127.0.0.1:8000> for chat, <http://127.0.0.1:8000/evaluations> for the eval dashboard.

See `EVALUATION.md` for the evaluation framework details.
